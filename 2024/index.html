Lingkungan Pengembangan Web

alat, kerangka kerja, library yang akan saya gunakan untuk melakukan perubahan pada situs web ini adalah

  • Tailwind CSS

    tailwind merupakan framework css yang akan saya gunakan untuk memperbagus UI dari blog ini, karena lebih mudah dan cepat bila dibandingkan dengan CSS Native. saya telah mempelajarinya selama periode 1 tahun lalu.

  • jQuery

    jquery merupakan library untuk pengembangan web, akan saya gunakan untuk membuat sebuah fungsi skrip, agar website jadi lebih interaktif. jQuery merupakan library yang berdiri di atas Javascript.

  • Local Storage

    dan yang terakhir, kemungkinan website ini mampu menyimpan data. misalkan to-do list, dll di local storage bila di perlukan.

  • Library Javascript

    saya belum bisa menentukan apa saja, namun saat ini saya akan mencoba beberapa library Javascript yang telah saya pelajari seperti P5.js dan Leaflet.js
