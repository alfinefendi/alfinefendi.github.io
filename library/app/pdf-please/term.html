Privacy Policy — PDF Please

Last updated: 2025

PDF Please is a simple document reader focused on privacy and ease of use. The application does not collect, store, or share any personal data from users in any form.

  • Information Collected

    PDF Please does not collect personal information. The application does not request, process, or transmit any data to external servers.

  • Access to Files

    PDF Please only opens PDF files that are explicitly selected by the user from their device. The application

    Does not upload files to any server

    Does not analyze file contents externally

    Does not store copies of files

    All processing occurs entirely on the user’s device.

  • Children Under 13

    The application can be used by all ages, including users under 13, because it does not collect personal data.

  • Data Sharing

    The application does not share any data with third parties.

  • Security

    Although the application does not transmit data, all operations remain local and secure. There is no data transfer over the internet.
